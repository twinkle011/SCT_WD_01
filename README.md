# 🌐 Responsive Landing Page

## 📖 Overview

This project was developed as part of the SkillCraft Technology Web Development Internship (Task 01).

A modern and responsive personal portfolio website built using HTML, CSS and JavaScript. The website features a fixed navigation bar, smooth scrolling, interactive hover effects and a clean user-friendly interface.

## ✨ Features

* Responsive Design
* Fixed Navigation Menu
* Smooth Scrolling Navigation
* Interactive Hover Effects
* Skills Section
* Education Section
* Projects Section
* Contact Section
* Modern UI Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser.

## 📂 Project Structure

* `index.html` → Main webpage
* `style.css` → Styling file
* `Twinkle.jpg` → Profile image
